System Upgrade OS/2 to NT 4.0 TS

System Manger, Project coordinator

Background

The so called "Store System" used at customers sites in former eastern european countries, needed to be exchanged with the overall common standard system called "MHS". That system was originally used as a terminal application but upgraded to the modern usage of GUI (Graphical User Interface) application. In order to provide the suitable infrastructure, OS/2, as server operating system, had to be exchanged with Windows NT 4.0 Terminal Server edition and Citrix Metaframe XP. Each Business Area in the store itself had to be equipped with components like Thin Clients, accompanied software like Network Station Manager and additional printers to guarantee the store to sell its products without disturbance.

Scope of transformation

The backoffice of each store, running OS/2 and the "Store System", was equipped with new server hardware and the compatible operating system Windows NT 4.0. Some installations had to be done during night, since customer's store normally opened at 09:00 hours every day and closes some time around 19:00 or 20:00 hours in the evening. In the time window of approx. 09:00 hours the new infrastructure was installed. New installation of Thin Clients and additional printers, removal of old workstations, on all Salesdesks in the store.

Scope of my deliverables

  • Coordinate the installations with local store IT
  • Installing and setup of new infrastructure together with the local store IT
  • Install new applications for stores to be used via Citrix on the Terminal Server
  • Introduce prepared incident management for the newly installed infrastructure
