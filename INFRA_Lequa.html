Active Directory Deployment

IT Infrastructure Architect, Technical specialist

Background

The customer's system environment was mainly based on Unix related infrastructure combined with an Active Directory run in non production mode. Business requirements made it necessary to establish a proper Active Directory Domain including SQL Server and different application servers.

Scope of transformation

Dimensioning, planning and implementing of a new Active Directory Domain at customer premises. The domain should contain a functioning Active Directory with two Domain Controllers, one SQL server and one application server to start with.

Scope of my deliverables

  • Dimensioning of hardware and software to enable proper domain setup including all required servers
  • Setup of floating licence server for Microsoft operating systems
  • User migration from a local user to domain users
  • SQL server 2008 installation, including backup and database configuration for applications
  • Installation of HP Quality Center
  • Setup of Kaspersky Endpoint Protection, network agent and deployment to clients
  • Migration of Windows 7 clients to new Active Directory Domain

  • Technology used: Windows Server 2012, SQL Server 2008, HP Quality Center, Kaspersky Antivirus Endpoint Protection

  • Communication: internal stakeholders, Microsoft, HP
