Business application compatibility

Project Leader

Background

In order to implement the next version of common platform running on Windows 7, all applications used by the customer needed to be verified for compatibility with the 64bit system architecture. The project had the task to secure the application compatibility 64bit system architecture and at the same time the consolidation of applications. The customer needed to secure about 60+ applications. Among those applications, Microsoft Office was one of the very important. It received special attention due to the many dependencies on other applications used at customers enterprise. The outcome of this project was to be seen as input to upcoming lifecycle activities within the enterprise.

Scope of transformation

With the main objective to deliver a report on standard and 3rd party applications that are compatible/not compatible with the 64bit system architecture, the project was divided into 2 Phases.

Phase 1 included:

  • Report of all compatible and non-compatible applications
  • Recommendation on which applications should be verified and transferred to Phase 2
  • Recommendation of consolidation of products providing the same or similar functions, which would ideally lead to a reduction of applications.
  • Listed dependencies on MS Office 2003
  • Technical description of the automated MS Office package
  • Test installations & transition preparations
  • Readiness for implemenation of MS Office

Phase 2 included:

  • Evaluation on conclusion from phase 1
  • Transformation of non-compatible applications together with business owners and product area managers

Scope of my deliverables

  • Planning & preparing application tests
  • Composing technical documentation and reports
  • Licence management handling in cooperation with licence manager
  • Communication & cooperation with external software suppliers & IT companies to secure compatibility
  • Identification of business applications having dependencies on Microsoft Office
  • Technical setup descriptions of applications
  • Test installation and transition preparation
  • Technical documentation of business applications
