Windows 10 Rollout Preparation

Project Manager

Background

Windows 10 is the latest version of Microsoft's operating system for PC computer, tablets, kiosks, and smartphones. With Windows 10, Microsoft changed its release management concept by introducing operating system as a service aka Windows as a Service (WaaS). Future updates of features and functionality of the operating system will be received through an automated push to the clients supported by the customer's common platform. To be able to successfully run a new Windows platform, project planning and application testing had to be performed.

Scope of transformation

Application testing and project roadmap creation for Windows 10 migration in 10 customer premises hosting about 900 computers worldwide.

Scope of my deliverables

  • Plan, evaluate, guide and steer application testing on Windows 10 platform for customer's financial applications
  • Create a roadmap for planning the migration from Windows 7 to Windows 10
  • Secure Windows 10 Infrastructure readiness
  • Secure testenvironment
  • Create an applicaton inventory with service owners and solution owners
  • Prepare for application testing
